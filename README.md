# nossky
Nostalgia Skyline - A recreation of an old screensaver on our home computer when I was a kid. 

Uses pygame - you can install it via `pip` with the command `pip install -r requirements.txt`

### Keyboard Shortcuts
Key | Function
--- | ---
`q` | quit
`f` | toggle fullscreen
`r` | reset skyline
